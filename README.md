ScorpioMessage
==============

## 协议生成工具，最简单的紧凑结构，目前可以转成cs,java,sco 用 Scropio-CSharp 脚本 作为配置
## Scropio-CSharp 传送门 https://github.com/qingfeng346/Scorpio-CSharp
## sco可以作为脚本实现协议热更新

## 把CSLib下的所有文件添加到自己的项目 如果不使用sco脚本可以删除 MessageSerializer 文件 把生成cs文件添加到项目即可使用
## java版本同上
